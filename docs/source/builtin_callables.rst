Builtin Callables
=================

Module :mod:`~automate.callables.builtin_callables` provides classes that may be used to various purposes
in Automate :class:`~automate.program.Program`, in condition and action attributes.
They are loaded automatically into :mod:`automate.callables` along with callables from possible installed extensions.
